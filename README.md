**1. Celsius to Fahrenheit formula**

F = (C * 1.8) + 32
--
36.7°C = 98.06000°F

---

**2. Fahrenheit to Kelvin formula**

K = ((F - 32) / 1.8) + 273.15
--
98.1°F = 309.8722K

---

**3. Kelvin to Rankine formula**

R = ((K - 273.15) * 1.8) + 491.67
--
309.9K = 557.8200ºR

---

**4. Rankine to Celsius formula**

C = (R - 491.67) / 1.8
--
557.8R = 36.73889C

---

**5. Celsius to Rankine formula**

R = (C * 1.8) + 491.67
--
100°C = 671.6700ºR
